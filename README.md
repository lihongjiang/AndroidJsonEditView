## 界面描述

[](https://raw.githubusercontent.com/lihongjiang/AndroidJsonEditView/master/doc/page.png)


	 1 四个一级功能，插入根数组；插入根对象；校验数据；复制到剪切板
	 2 标签对象操作：添加对象，添加数组，删除标签，添加属性,层级收缩，key有无
	 3 属性对象操作：删除
	 4 单击key/vaule修改对应值
